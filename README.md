RecyclAI - Smart Waste Management

♻️ What is it?

RecyclAI is an AI-powered smart bin that physically prevents recycling contamination. By using Computer Vision on a Raspberry Pi, it identifies the type of waste held by the user and automatically opens the correct compartment, while logging the activity to a live web dashboard.

🛠️ Technologies Used

Artificial Intelligence: Python, TensorFlow, Keras (MobileNetV2 CNN)

Web Stack: Java (Spring Boot), React.js, Material UI, WebSockets

Integration & Hardware: Node-RED, Raspberry Pi 4, Servo Motors

DevOps & OS: GitLab CI/CD, SonarQube, Java Multithreading

✨ Key Features

Edge AI Classification: Real-time image processing on a Raspberry Pi achieving >90% accuracy in waste classification.

Digital Twin OS Simulation: A multi-threaded Java simulation applying OS concepts (Monitors, Mutexes, and Message Passing) to solve the Producer-Consumer problem and prevent deadlocks/race conditions.

Live Dashboard: A React Single Page Application providing facility managers with real-time video feeds and activity logs via WebSockets.

Automated CI/CD: GitLab pipelines configured with Playwright (E2E testing), JUnit, and SonarQube quality gates.
